## TrayStatus 是什么，解决什么问题

它是 Binary Fortress Software 出品的一款 Windows 系统托盘增强工具。说白了，就是在右下角任务栏托盘区域，实时显示几个键盘按键和系统资源的状态图标。

软件本身设计得很克制——不弹窗、不打扰、不吃资源，就安安静静趴在那里。你随时瞄一眼，一切都知道了。

### 键盘状态显示

支持在托盘里显示：

- **Caps Lock**（大写锁定）
- **Num Lock**（数字锁定）
- **Scroll Lock**（滚动锁定）
- **Alt / Ctrl / Shift / Win 键**状态

对于码字、做 Excel、打代码的人来说，这几个状态一眼明，能省不少心。

### 系统性能监控

Pro 版解锁了 CPU、内存、硬盘的实时状态指示器，还可以通过性能计数器监测更细的指标，比如网络活动、蓝牙状态、磁盘队列长度等。

不用专门打开任务管理器，托盘一瞥，系统当前状态心中有数。

### 触发器（Pro 版专属）

这是个特别实用的功能：你可以设置在某个特定程序里，自动启用或禁用 Caps Lock / Num Lock / Scroll Lock。

比如切到 Excel 时自动开启数字锁定，切回 Word 时自动关掉——完全不用手动切换。

### 双击快速切换 + 按键提示音

双击托盘里的按键图标，直接切换对应键的状态，不用伸手摸键盘。切换时还可以播放提示音，不用看屏幕也能感知状态变化，适合键盘灯不亮的机械键盘用户。

### 个性化图标主题

内置浅色、深色等多套图标风格，硬盘指示器的文字颜色、阴影也可以自定义。强迫症友好。
